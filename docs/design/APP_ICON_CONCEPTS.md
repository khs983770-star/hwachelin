# 화슐랭 앱 아이콘 컨셉 3안

> 작성: 디자이너 / 2026-05-27
> 목적: 현재 앱 아이콘의 정보 과잉 문제 개선

## 현재 아이콘의 문제

- 변기 아이콘 + "화슐랭" + "HWA-CHELIN" + 매화꽃 → 60×60pt 작은 사이즈에서 모두 뭉개짐
- iOS 홈스크린 / Android 런처에서 식별성이 떨어짐
- 텍스트는 앱 이름 라벨이 따로 보여주므로 아이콘 안에 들어갈 필요 없음

## 공통 가이드라인

| 항목 | 값 |
|---|---|
| 캔버스 | 1024 × 1024 px (마스터) |
| 안전 영역 | 중앙 80% (코너에 콘텐츠 X) |
| 모서리 | iOS는 자동 라운드, 디자인은 사각으로 |
| 색상 | `#E51B3E` (brand 500) 기반 |
| 보조 색상 | `#FFFDFB` (neutral 50), `#7E0719` (brand 800) |
| 텍스트 | 절대 X (앱 이름 라벨이 따로 보임) |

## 컨셉 A — "매화 핀" (브랜드 추천)

### 아이디어
- 매화꽃 한 송이를 지도 핀 모양으로 변형
- 화슐랭의 "꽃 = 별점" 컨셉을 그대로 아이콘에
- 추상적이고 우아함, 미슐랭 가이드 분위기

### 비주얼
```
배경:      그라데이션 #E51B3E → #C8112F (45도)
중앙:      흰색 매화꽃 (5장 꽃잎, 라운드)
           아래쪽 한 꽃잎을 길게 늘려 핀 꼬리 형태
하이라이트: 꽃 중앙에 #FFFDFB 작은 동그라미
```

### SVG 시안
```svg
<svg viewBox="0 0 1024 1024" xmlns="http://www.w3.org/2000/svg">
  <!-- 배경 -->
  <defs>
    <linearGradient id="bg" x1="0" y1="0" x2="1" y2="1">
      <stop offset="0%" stop-color="#E51B3E"/>
      <stop offset="100%" stop-color="#C8112F"/>
    </linearGradient>
  </defs>
  <rect width="1024" height="1024" fill="url(#bg)"/>

  <!-- 매화 5장 + 핀 꼬리 -->
  <g transform="translate(512 460)" fill="#FFFDFB">
    <!-- 꽃잎 5장: 72도 간격, 위쪽부터 -->
    <circle cx="0"    cy="-160" r="115"/>
    <circle cx="152"  cy="-49"  r="115"/>
    <circle cx="94"   cy="129"  r="115"/>
    <circle cx="-94"  cy="129"  r="115"/>
    <circle cx="-152" cy="-49"  r="115"/>
    <!-- 핀 꼬리 (아래로 늘어남) -->
    <path d="M -60 200 Q 0 360 60 200 Z"/>
  </g>

  <!-- 중앙 동그라미 (꽃술) -->
  <circle cx="512" cy="460" r="48" fill="#E51B3E"/>
</svg>
```

### 장점
- 단순, 강한 인지도
- 작은 사이즈에서도 형태 식별 가능
- 미슐랭 가이드 같은 "고급" 분위기

### 단점
- 화장실 앱이라는 직관성이 약함 → 핀 꼬리로 보완

## 컨셉 B — "변기 미니멀" (직관성 추천)

### 아이디어
- 변기 심볼만 남기고 모든 텍스트 제거
- 라운드 처리된 미니멀 변기
- 한눈에 화장실 앱임을 알 수 있음

### 비주얼
```
배경:      단색 #E51B3E
중앙:      흰색 변기 심볼 (라운드 처리, 두꺼운 라인)
           심볼 위 작은 ✿ 한 송이 (#FFD700 또는 #FFFDFB)
```

### SVG 시안
```svg
<svg viewBox="0 0 1024 1024" xmlns="http://www.w3.org/2000/svg">
  <rect width="1024" height="1024" fill="#E51B3E"/>

  <!-- 변기 본체 (위쪽 탱크 + 아래쪽 변기) -->
  <g fill="#FFFDFB">
    <!-- 탱크 -->
    <rect x="350" y="320" width="220" height="180" rx="24"/>
    <!-- 시트 -->
    <ellipse cx="512" cy="600" rx="200" ry="80"/>
    <!-- 바닥 -->
    <rect x="380" y="640" width="280" height="80" rx="20"/>
    <rect x="430" y="700" width="180" height="40" rx="12"/>
  </g>

  <!-- 매화 한 송이 (오른쪽 위) -->
  <g transform="translate(720 260) scale(0.5)" fill="#FFFDFB">
    <circle cx="0"    cy="-60" r="42"/>
    <circle cx="57"   cy="-19" r="42"/>
    <circle cx="35"   cy="48"  r="42"/>
    <circle cx="-35"  cy="48"  r="42"/>
    <circle cx="-57"  cy="-19" r="42"/>
    <circle cx="0"    cy="0"   r="18" fill="#E51B3E"/>
  </g>
</svg>
```

### 장점
- 직관성 최고 — "화장실 앱"임이 즉시 보임
- 단순해서 작은 사이즈에서도 인식 가능

### 단점
- 다른 화장실 앱과 차별화가 적음
- "미슐랭 가이드" 같은 고급 분위기는 약함

## 컨셉 C — "H 모노그램" (앱스토어 검색 친화)

### 아이디어
- "화슐랭"의 ㅎ 자음 또는 H를 매화꽃 + 핀 형태로 디자인
- 텍스트는 없지만 알파벳 형태로 인지
- 모노그램 스타일 (Airbnb, Slack 등)

### 비주얼
```
배경:      그라데이션 #FFFDFB → #FFE0E5 (베이지 → 옅은 핑크)
중앙:      커다란 #E51B3E "ㅎ" 또는 "H"
           세로 획을 핀 모양으로, 가로 획에 매화 꽃잎 5장
```

### SVG 시안
```svg
<svg viewBox="0 0 1024 1024" xmlns="http://www.w3.org/2000/svg">
  <defs>
    <linearGradient id="bg2" x1="0" y1="0" x2="0" y2="1">
      <stop offset="0%" stop-color="#FFFDFB"/>
      <stop offset="100%" stop-color="#FFE0E5"/>
    </linearGradient>
  </defs>
  <rect width="1024" height="1024" fill="url(#bg2)"/>

  <!-- H 형태 (좌·우 세로 기둥 + 중간 가로) -->
  <g fill="#E51B3E">
    <!-- 왼쪽 기둥 (핀 모양: 위는 둥글고 아래는 뾰족) -->
    <path d="M 280 240
             Q 280 200 320 200
             L 360 200
             Q 400 200 400 240
             L 400 720
             Q 400 800 340 800
             Q 280 760 280 720 Z"/>
    <!-- 오른쪽 기둥 -->
    <path d="M 624 240
             Q 624 200 664 200
             L 704 200
             Q 744 200 744 240
             L 744 720
             Q 744 800 684 800
             Q 624 760 624 720 Z"/>
    <!-- 가로 획 -->
    <rect x="400" y="460" width="224" height="80" rx="12"/>
  </g>

  <!-- 가로 획 위에 매화 5장 -->
  <g transform="translate(512 500) scale(0.45)" fill="#FFFDFB">
    <circle cx="0"    cy="-60" r="42"/>
    <circle cx="57"   cy="-19" r="42"/>
    <circle cx="35"   cy="48"  r="42"/>
    <circle cx="-35"  cy="48"  r="42"/>
    <circle cx="-57"  cy="-19" r="42"/>
    <circle cx="0"    cy="0"   r="14" fill="#E51B3E"/>
  </g>
</svg>
```

### 장점
- 차별화 — 다른 화장실 앱과 완전히 다른 분위기
- 베이지 배경이라 빨강 일변도 홈스크린에서 눈에 띔
- 매화 + H 결합이 영리하게 느껴짐

### 단점
- 직관성 떨어짐 — 무슨 앱인지 한눈에 모름
- 한글/영문 둘 다 인식해야 의도 전달

## Export 가이드 (무료 도구)

### Figma (추천)
1. 위 SVG 코드 → Figma에 직접 paste (Frame → Paste 자동 import)
2. 1024×1024 캔버스
3. Export → PNG 1× (1024) / 또는 다양한 사이즈
4. `icon.png` (1024)을 `assets/icon.png`에 교체

### Canva (더 쉬움)
1. 1024×1024 캔버스 생성
2. 위 비주얼 가이드 참고하여 도형 조합
3. PNG export

### 필요한 아이콘 사이즈 (app.json 참고)
| 파일 | 사이즈 | 용도 |
|---|---|---|
| `icon.png` | 1024×1024 | iOS 메인 |
| `adaptive-icon.png` | 1024×1024 (안전영역 66%) | Android |
| `splash-icon.png` | 1242×2436 | 스플래시 |
| `favicon.png` | 48×48 | 웹 |
